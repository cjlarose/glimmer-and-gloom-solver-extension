## Glimmer & Gloom Solver Extension

Browser extension for helping players solve Glimmer and Gloom games on Flight Rising

## Build

```sh
pnpm install
pnpm build
```

To build the browser extension zip

```sh
pnpm zip
```

## Watch

```sh
pnpm watch
```

## Preview in browser

```sh
script/launch
```

## Set up git hooks

```sh
git config core.hooksPath .githooks
```
